<h1 align="center">Laravel Backend/API</h1>

## Installation
- Install Docker
- Clone the repository with __git clone__
- `./vendor/bin/sail up`
- `sail artisan breeze:install`
- `npm install && npm run dev`
- `sail artisan migrate`
- `sail artisan db:seed`
